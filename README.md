Tool to convert string

## Install

With Go version `>= 1.16`:

```sh
go install github.com/haunt98/converter@latest
```

With Go version `< 1.16`:

```sh
GO111module=on go get -u github.com/haunt98/converter
```

## Usage

```sh
converter "This a a text"
```
